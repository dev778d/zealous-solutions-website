Copy and paste these commands one by one:

```bash
git remote add origin https://github.com/YOUR_USERNAME/zealous-solutions-website.git

```

Replace `YOUR_USERNAME` with your actual GitHub username.

### 3. Enable GitHub Pages
1. In your new repository, click "Settings" tab
2. Scroll to "Pages" in left sidebar
3. Under "Source": Select "Deploy from a branch"
4. Choose "main" branch and "/docs" folder
5. Click "Save"

**Your website will be live at:** `https://YOUR_USERNAME.github.io/zealous-solutions-website/`

### 4. Connect Your GoDaddy Domain
1. **In GitHub Pages Settings**:
   - Add custom domain: `zealous-solutions.com`
   - Check "Enforce HTTPS"

2. **In GoDaddy Control Panel**:
   - Go to DNS Management
   - Set up URL Forwarding:
     - From: `zealous-solutions.com`
     - To: `https://YOUR_USERNAME.github.io/zealous-solutions-website/`
     - Forward type: "Permanent (301)"
     - Settings: "Forward with masking"
